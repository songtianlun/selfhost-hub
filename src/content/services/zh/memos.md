# Memos

Memos 是一个开源的、自托管的备忘录和知识管理工具。它支持 Markdown 格式，提供标签系统，可以帮助你快速记录想法、整理知识。

## 主要功能

- **笔记功能**：
  - Markdown 支持
  - 标签系统
  - 时间线视图
  - 搜索功能
- **用户系统**：
  - 多用户支持
  - 用户权限管理
  - 个人设置
- **其他功能**：
  - API 支持
  - 数据导入导出
  - 自定义主题
  - 响应式设计

## 技术特点

- 使用 Go 语言开发后端
- 使用 React 开发前端
- 使用 SQLite 数据库
- 支持 Docker 部署
- 提供 REST API

## 部署要求

- 支持 Docker 的环境
- 或 Go 1.16 或更高版本
- 建议使用 HTTPS
- 足够的内存（建议至少 512MB）
