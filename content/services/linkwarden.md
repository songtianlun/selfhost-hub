LinkWarden 是一个自托管、开源的协作式书签管理器,旨在收集、阅读、标注和完整保存您认为重要的内容,并将所有功能集于一处。它通过自动保存网页截图、PDF 和 HTML 文件来解决链接失效问题,兼具稍后阅读的简洁性和网页归档的可靠性。

## 主要功能

- **自动归档**:为每个网页自动捕获截图、PDF 和单一 HTML 文件
- **阅读器视图**:提供文本高亮和标注工具
- **Wayback Machine 集成**:可选的网页快照功能
- **AI 自动标签**:基于本地 AI 的智能内容标签
- **层次化组织**:支持集合、子集合和标签的多层次管理
- **多用户协作**:可自定义成员权限的团队协作功能
- **分享功能**:支持公开链接分享和集合分享
- **强大搜索**:全文搜索、过滤和排序功能
- **渐进式 Web 应用**:支持 PWA 功能
- **浏览器扩展**:支持浏览器扩展和 Floccus 同步
- **导入导出**:支持书签的导入和导出
- **主题切换**:支持暗色/亮色模式
- **RSS 订阅**:支持 RSS 源订阅
- **API 访问**:提供 API 密钥访问
- **批量操作**:支持批量操作
- **多语言支持**:支持国际化(i18n)

## 技术栈

- TypeScript (90.5%)
- React/Next.js 框架
- JavaScript
- Yarn 包管理
- Docker 容器化部署
- Monorepo 架构

## 部署要求

- 支持 Docker 容器部署
- 需要配置 .env 环境文件
- 支持企业 SSO 集成
- 建议配置:2核 CPU,2GB 内存以上
- 存储空间:根据归档内容量而定,建议至少 10GB
- 网络:需要访问互联网以归档网页内容

## 特色优势

- **内容保存**:通过多格式捕获确保长期内容可访问性
- **协作友好**:内置多用户功能和精细权限控制
- **数据主权**:自托管选项确保完全控制您的数据
- **活跃社区**:15.5k+ GitHub 星标,活跃的 Discord 社区
- **跨平台生态**:社区构建的 iOS、macOS 和 Android 应用

## 使用场景

- 个人知识库管理和归档
- 团队协作的资源收集和分享
- 学术研究的资料收集和标注
- 防止重要网页链接失效
- 构建私有的网页归档库
