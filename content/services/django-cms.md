Django CMS 是一个基于 Django 框架开发的企业级内容管理系统，提供了强大的内容编辑和管理功能，特别适合构建企业网站和大型内容平台。

## 主要功能

- **拖拽编辑**：直观的拖拽式页面编辑
- **多语言支持**：完整的多语言内容管理
- **插件系统**：丰富的插件生态
- **版本控制**：内容版本管理
- **权限管理**：细粒度的用户权限
- **SEO 工具**：内置 SEO 优化功能
- **API 支持**：REST API 接口

## 部署要求

- Python 3.8 或更高版本
- Django 3.2 或更高版本
- PostgreSQL/MySQL
- 推荐配置：2核 CPU，4GB 内存
- 存储空间：根据项目规模，建议 10GB 以上
