Medama 是一个注重隐私的网站分析平台，提供了现代化的用户界面和完整的数据分析功能，同时确保用户数据的安全性和隐私性。

## 主要功能

- **隐私优先**：内置隐私保护机制
- **现代化界面**：直观的数据可视化
- **实时分析**：实时访问数据统计
- **多维度分析**：访问来源、设备、地区等分析
- **自定义事件**：支持自定义事件跟踪
- **数据导出**：支持多种格式导出
- **API 支持**：提供完整的 API 接口
- **多语言支持**：支持多语言界面

## 部署要求

- Node.js 环境
- 支持 Docker 部署
- 需要数据库支持
- 支持反向代理
- 建议使用 HTTPS
- 支持云服务部署
