CommaFeed 是一个轻量级的自托管 RSS 阅读器，采用 Java 开发。它提供了类似 Google Reader 的简洁界面，适合个人使用。

## 主要功能

- **简洁界面**：类似 Google Reader 的简洁界面
- **移动端支持**：响应式设计，支持移动设备
- **导入导出**：支持 OPML 导入导出
- **多用户**：支持多用户系统
- **标签系统**：支持文章标签管理
- **搜索功能**：支持文章搜索
- **快捷键**：提供键盘快捷键支持
- **REST API**：提供 API 接口
- **主题支持**：支持自定义主题

## 部署要求

- Java 8+ 环境
- PostgreSQL 数据库
- 最低配置：1核 CPU，256MB 内存
- 推荐配置：2核 CPU，512MB 内存
- 存储空间：约 100MB（不包括数据库数据）
