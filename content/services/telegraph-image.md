Telegraph-Image 是一个基于 Cloudflare Pages 的免费图床服务，它利用 Telegraph 的 API 来存储图片，无需自己的服务器即可部署。这是一个轻量级的解决方案，适合个人使用。

## 主要功能

- **免费部署**：基于 Cloudflare Pages，无需服务器
- **简单易用**：支持拖拽上传和粘贴上传
- **图片管理**：支持图片删除和查看
- **链接格式**：提供多种链接格式（HTML、BBCode、Markdown 等）
- **无限制**：没有存储空间和流量限制
- **快速访问**：利用 Cloudflare CDN 加速
- **开源免费**：完全开源，可自由部署

## 部署要求

- Cloudflare 账号
- 基本的 Git 知识
- 无需服务器
- 无需数据库
- 无需特殊配置
- 部署时间：约 5 分钟
