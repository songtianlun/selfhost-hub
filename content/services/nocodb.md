NocoDB 可以把 MySQL、PostgreSQL、SQL Server 等关系型数据库转换成类似 Airtable 的在线协作表格，帮助团队在无需写代码的情况下搭建内部系统与数据工作流。

## 核心功能

- **多数据库支持**：兼容主流 SQL 数据库，也可连接现有业务库直接生成表格视图。
- **丰富的视图类型**：提供表格、看板、画廊、日历、甘特图等不同视图，满足多场景需求。
- **角色与权限**：支持多种权限级别，并能针对视图或字段设置访问范围，保障数据安全。
- **自动化与集成**：内置自动化流程，可连接 Slack、Teams、Zapier 等服务触发通知或同步。
- **API 网关**：自动生成 REST 与 GraphQL API，便于快速对接前端或第三方系统。
- **表单收集**：快速生成对外分享的 Web 表单，自动落地到数据库表中，适合线索收集与问卷场景。
- **插件生态**：可通过 Marketplace 安装扩展组件，增强图表分析、数据导入导出等高级功能。

## 典型场景

- **业务团队自助建表**：销售或运营团队可在现有业务数据库之上搭建 CRM、线索池等协作表格。
- **项目流程管理**：结合看板、甘特图视图与通知，覆盖项目进度追踪、排程与资源协调。
- **数据门户与仪表盘**：利用图表与视图共享，把复杂 SQL 数据转换成面向业务方的易读界面。

## 部署建议

1. 使用官方 Docker 镜像即可快速启动，推荐搭配外部数据库以便升级与备份。
2. 在 `docker-compose.yml` 中配置 `NC_DB` 相关环境变量，指向持久化数据库实例，并设置 `NC_PUBLIC_URL` 供系统生成正确的链接。
3. 生产环境建议启用反向代理与 HTTPS，配置 SMTP 邮件服务（`NC_SMTP_FROM`、`NC_SMTP_USER` 等）用于邀请协作者。
4. 如果需要存储大量附件，可选择对象存储（如 S3、MinIO），并在环境变量中启用相关驱动。
5. 定期备份数据库与上传的静态资源，确保视图配置和文件附件安全。
