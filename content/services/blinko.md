Blinko 是一款开源的自托管第二大脑，聚合笔记、任务与灵感捕获，结合 AI 辅助能力帮助个人和团队整理信息、快速回顾并持续迭代知识库。

## 核心功能

- **统一知识库**：支持以 Markdown 记录笔记，按标签与双向链接组织内容，便于从不同视角浏览关联信息。
- **AI 辅助**：在笔记中调用大模型进行摘要、问答或改写，快速提炼要点或获取上下文建议（需配置兼容的 AI API）。
- **任务与闪念收集**：提供待办与闪念入口，将零散想法与行动项集中到同一工作台。
- **搜索与时间线**：全局搜索搭配时间线视图，方便回溯历史记录或定位近期更新。
- **多端访问**：基于 Web 的响应式界面，适配桌面与移动浏览器，便于自建内网或公网入口。

## 部署建议

1. 建议使用 Docker Compose 运行应用容器，并为数据、上传与配置文件挂载持久化卷，避免升级或重启导致数据丢失。
2. 如需开启 AI 能力，在 `.env` 或容器环境变量中配置 OpenAI 兼容的 API 域名与密钥，确认网络出口可访问相应模型服务。
3. 通过 Nginx/Caddy 等反向代理为前端提供 HTTPS 访问，并在可信的存储设备上定期备份数据卷，确保知识库与模型配置安全。
