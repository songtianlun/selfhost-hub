Floccus is a powerful browser extension that enables you to sync your native browser bookmarks privately across multiple browsers and devices. It supports various backend services including Nextcloud Bookmarks, Linkwarden, KaraKeep, Google Drive, Git servers, and WebDAV-compatible services.

## Key Features

- **Cross-browser Sync**: Compatible with Firefox, Chrome, Edge, Opera, Brave, and Vivaldi
- **Multiple Backend Support**: Works with Nextcloud Bookmarks, Linkwarden, KaraKeep, Google Drive, Git (GitHub/GitLab/Gitea), and WebDAV
- **Mobile Apps**: Available for Android and iOS
- **Flexible Sync Strategies**: Supports uni-directional or bi-directional synchronization
- **Multiple Profiles**: Create multiple sync configurations for different use cases
- **Customizable Sync Intervals**: Adjust sync frequency and select specific folders
- **Configuration Export**: Export and import your settings
- **Privacy-focused**: Local encryption with full control over your data

## Technology Stack

- TypeScript/JavaScript
- Node.js
- Webpack and Gulp build tools
- Browser extension framework (Manifest V3 compatible)
- Android Studio (for mobile apps)

## Deployment Requirements

- Modern browser (Firefox, Chrome, Edge, etc.)
- Optional: Self-hosted backend service (Nextcloud, WebDAV server, etc.)
- Mobile: Android 5.0+ or iOS 13+
- Network: Access to your chosen backend service

## Use Cases

- Sync bookmarks across multiple browsers
- Protect privacy using self-hosted services
- Share bookmark collections with teams
- Access bookmarks across devices including mobile
