- Serves static files with optional uploads, folder drag-and-drop, and resumable transfers.
- Provides user authentication, HTTPS, and CORS controls for secure sharing.
- Built-in search, inline file editing, ZIP downloads, and health checks.
- Ships a Docker image or can be compiled directly with Rust for lightweight deployments.
